# Vue 购物平台

这是一个使用 Vue.js 构建的电子商务平台应用程序，具备商品展示、搜索、购物车、下单支付等完整电商功能。项目结构清晰，适合学习 Vue.js 的实际应用开发。

## 项目设置

```bash
npm install
```

## 开发环境运行

```bash
npm run serve
```

## 构建生产环境版本

```bash
npm run build
```

## 代码检查和修复

```bash
npm run lint
```

## 自定义配置

有关配置的更多信息，请参阅 [Vue CLI 配置文档](https://cli.vuejs.org/config/).

## 功能特点

- 商品详情展示与图片放大预览
- 首页轮播图与楼层展示
- 商品搜索与筛选功能
- 用户登录与注册
- 购物车管理
- 下单与支付流程
- 订单管理与查看

## 目录结构

- `public/` 静态资源
- `src/assets/` 项目图片资源
- `src/components/` 可复用的 Vue 组件
- `src/pages/` 页面组件
- `src/store/` Vuex 状态管理模块
- `src/router/` Vue Router 配置
- `src/api/` 接口请求模块
- `src/mock/` 模拟数据与接口
